Téléchargement du projet :

- git clone https://github.com/cyrilmanuel/IAclassificationTP.git
- cd IAclassificationTP

installation : 
aller dans le dossier IAclassificationTP.

commencer par créer un environnement virtuel "venv" et utiliser le.

- virtualenv venv
- cd venv
- source /bin/activate

ensuite installer les dépendences du fichier requirement.txt

- pip install -r requirement.txt

ensuite executer le fichier SeparatedTrainingTest.py

- python SeparatedTrainingTest.py

puis executer le fichier Classification.py

- python Classification.py
